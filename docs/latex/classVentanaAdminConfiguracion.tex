\-Ventana del admin para configurar la aplicación.

\-Se compone de una lista de \-Tab\-Admin\-Config$\ast$, que al momento de clickear el botón \char`\"{}\-Aplicar\char`\"{}, es recorrida aplicando los cambios por pestaña.
